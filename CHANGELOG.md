# Changelog

All notable changes to the PWP Plugin will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-03-02

### Added
- **pwp-debug** — Systematic debugging protocol (reproduce → isolate → root cause → fix → verify)
- **pwp-code-review** — Structured code review with severity levels and checklist
- **pwp-security-audit** — Security review covering OWASP Top 10, secrets, auth, and data protection
- **pwp-refactor** — Safe refactoring protocol with patterns and safety verification
- **pwp-migrate** — Database and platform migration protocol with rollback strategies
- **pwp-api-design** — REST API design standards with conventions, status codes, and validation
- **pwp-perf** — Performance optimization protocol (measure → identify → fix → verify)
- **pwp-bootstrap** — Project initialization from empty directory to production-ready foundation
- **pwp-test** — Testing strategy covering test pyramid, naming, coverage targets, and data factories
- **pwp-deploy** — Deployment protocol with environment ladder, feature flags, and rollback plans
- **pwp-docs** — Documentation lifecycle with feature traceability, changelogs, and walkthroughs
